Markdown 简单介绍


Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

Markdown 语言在 2004 由约翰·格鲁伯(英语：John Gruber)创建。

Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。

Markdown 编写的文档后缀为 .md, .markdown。

编辑器推荐

编辑器推荐使用Typora，当然这个是软件都是付费的哈。

Markdown 段落

换行

markwown语法的换行有两种方式：

一、直接在要换行的地方敲两个空格。

展示效果：

我要换行啦！
我是零一行！

二、直接在要换行的地方敲两个回车。

展示效果：

我要换行啦！

我是零一行！

这两种换行的差距还是蛮大的，第一种方法相当于是另起一行，而第二种方法相当于是另起一段，主要是间距大小不一样哈，我这边还是喜欢敲两次回车（真的是太爽啦😍）。
